Projetos

  • Irene Festas

    Desenvolvimento de site responsivo utilizando HTML5, CSS3 e JavaScript, com abordagem mobile first, integração de animações via ScrollReveal e foco em UX/UI e conversão. Projeto real publicado em produção, com otimização para múltiplos dispositivos e canais de contato.

  • Jogo Da Velha

    Aplicação desenvolvida em JavaScript com lógica completa de jogo, incluindo alternância de jogadores, verificação automática de vitória ou empate e reinício de partida. O projeto demonstra domínio de lógica de programação e manipulação do DOM.

  • Cadastro De Usuários

    Aplicação web criada com React, integrada a uma API RESTful e banco de dados MongoDB, permitindo listar, cadastrar e excluir usuários. O projeto prioriza organização de código, comunicação com API via requisições HTTP e uma interface clara e responsiva.

  • Relógio Digital

    Relógio digital desenvolvido com JavaScript, exibindo horas, minutos e segundos em tempo real. O projeto utiliza atualização dinâmica do DOM e controle preciso de tempo, focando em simplicidade, desempenho e clareza visual.
