
# 💼 Portfólio Pessoal — Werlys Santos  
  
Site de portfólio pessoal desenvolvido para apresentar meus **projetos**, **habilidades técnicas** e **formas de contato**, com foco em desenvolvimento **Full Stack** e aplicações web modernas.  
  
O projeto foi construído com uma abordagem **mobile first**, design moderno e atenção à experiência do usuário (UX/UI).

## 🧠 Objetivo do Projeto  
  
- Centralizar informações profissionais em um único lugar  
- Apresentar projetos reais e autorais  
- Facilitar o contato com recrutadores e clientes  
- Demonstrar domínio de HTML, CSS e JavaScript puro  
  
---  
  
## 🚀 Funcionalidades  
  
- Navegação suave entre seções (scroll behavior)  
- Layout responsivo (mobile, tablet e desktop)  
- Menu mobile com botão hamburguer  
- Seção “Sobre mim” com apresentação profissional  
- Slider de projetos com navegação (Swiper.js)  
- Cards de projetos com descrição técnica  
- Formulário de contato integrado ao WhatsApp  
- Links diretos para LinkedIn, GitHub e e-mail  
  
---  
  
## 🛠 Tecnologias Utilizadas  
  
- **HTML5** — Estrutura semântica  
- **CSS3** —   
 - Flexbox   
 - Media Queries   
 - Animações e efeitos visuais   
 - Design Glassmorphism   
- **JavaScript (Vanilla)** — Interações e comportamento dinâmico  
- **Swiper.js** — Carrossel de projetos  
- **Font Awesome** — Ícones  
- **Google Fonts** — Tipografia personalizada  
---  
  
## 📱 Responsividade  
  
O site foi desenvolvido seguindo o conceito **mobile first**, adaptando-se corretamente a diferentes tamanhos de tela e dispositivos.
